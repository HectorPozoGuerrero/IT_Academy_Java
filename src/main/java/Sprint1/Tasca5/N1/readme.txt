INSTRUCCIONES PARA EJECUTAR EL PROGRAMA DESDE LA LÍNEA DE COMANDOS:

1. Abrir el cmd de Windows.

2. Una vez tengamos la ventana del cmd abierta, debemos introducir la
   palabra "java" seguida de un espacio en blanco y la ruta donde se
   encuentre el programa que queremos ejecutar. Por ejemplo:
   java C:\Users\Hecto\Documents\GitHub\IT_Academy_Java\src\main\java\Sprint1\Tasca5\N1\Ex1\Main.java

3. Pulsamos intro y el programa se ejecutará. En este caso, nos pide que
   introduzcamos la ruta del directorio que queremos que nos ordene.
   Por ejemplo: C:\Users\Hecto\Documents\GitHub\IT_Academy_Java\src\main\java\Sprint1\Tasca5

4. Pulsamos intro nuevamente y el programa nos devolverá el directorio
   ordenado alfabéticamente.
